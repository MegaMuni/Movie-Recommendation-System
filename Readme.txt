1. Download the movies dataset from the movies dataset website
(https://grouplens.org/datasets/movielens/latest/)
2. Run the first ipynb file (Movie_recommendation_preprocessing_and_EDA.ipynb)
this does some preprocessing, and mainly helps with exploratory data analysis to understand characteristics of the data
3. Run the second ipynd file (Movie_Recomendation_ALS.ipynb)
The file does hyperparameter tuning, trains the model and saves it
The last part includes prediction pipeline where we can give the userId whose recommendations we need
